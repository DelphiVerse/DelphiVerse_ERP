## 1. PARA COMPILAR O SERVER

### 1. PARA COMPILAR O SERVER

1 ** Abrir o Prompt de comando na pasta do projeto.**
2 ** Iniciar o BOSS:** Executar o comando boss init.
3 ** Instalar o Horse:** Executar o comando boss install horse.
4 ** Instalar o Jhonson:** Executar o comando boss install Jhonson.
4 ** Instalar o dataset-serialize:** Executar o comando boss install github.com/viniciussanchez/dataset-serialize.
